Stage A3 WEB

Janvier – Février 2026 6 semaines La Séguinière (49) Stage 2
L'entreprise

A3 WEB

A3 WEB est une agence web basée à La Séguinière, en Maine-et-Loire (49). Elle accompagne ses clients dans la création, la refonte et la maintenance de sites internet, ainsi que dans l'intégration de solutions digitales sur mesure. L'agence travaille principalement sous WordPress et propose des prestations de développement back-end, d'automatisation et d'intégration d'API tierces.

Durant ce stage de 6 semaines, j'ai intégré l'équipe technique et participé activement à plusieurs projets clients en cours, intervenant aussi bien sur la maintenance que sur le développement de nouvelles fonctionnalités et l'intégration d'API externes.

WordPress PHP API Slack API Wrike GitLab VS Code ACF
01

Maintenance WordPress

  • Mises à jour du core — mise à jour de WordPress et de ses extensions sur plusieurs sites clients en production
  • Corrections post-MAJ — identification et résolution des conflits de plugins et problèmes d'affichage apparus après les mises à jour
  • Suivi des procédures — respect des procédures internes de maintenance et rédaction de comptes-rendus d'intervention

Interface WordPress : tableau de bord des mises à jour du core et des extensions

02

Back-end — Automatisation de l'assignation des tickets

  • Script PHP d'assignation automatique — développement d'un script attribuant les tickets entrants au bon collaborateur selon des règles métier définies (type, produit, priorité)
  • Intégration dans WordPress — intégration via hooks PHP (functions.php) dans le système de tickets existant
  • Relance automatique J+7 — déclenchement automatique d'un e-mail de relance si un ticket reste sans réponse après 7 jours

Extrait du script PHP : logique d'assignation automatique des tickets selon les règles métier

03

Intégrations API — Slack & Wrike

  • Intégration API Slack — mise en place de notifications automatiques dans un canal Slack à chaque création ou mise à jour de ticket client
  • Intégration API Wrike — synchronisation des tickets entre le système client et l'outil de gestion de projet Wrike
  • Gestion des erreurs — implémentation d'un mode test pour rediriger les messages vers un destinataire de test sans affecter la production

Extrait du code : fonction d'envoi de message Slack avec gestion du mode test

04

Front-end & Refonte graphique

  • Corrections de mise en page — responsive, espacements et typographies sur différents sites clients
  • Zone d'alerte ACF — création d'un bloc d'alerte personnalisable via Advanced Custom Fields, permettant aux administrateurs de diffuser des messages d'incident en temps réel
  • Refonte graphique — participation à la refonte complète du site client : nouvelle identité visuelle, navigation repensée et interface modernisée

LOCATOUT

AP 3 · Gestion de la location de voitures · PHP natif + MySQL + XAMPP

Contexte du projet

LOCATOUT est un leader européen de la location de voitures, présent dans plus de 140 pays grâce à un solide réseau d'agences. Pour évaluer l'activité de son service après-vente, la direction a décidé d'informatiser complètement la gestion des locations de voitures et fait appel à notre équipe pour étudier l'évolution de son système informatique.

Réalisé en équipe dans le cadre du TP E6 (BTS SIO SLAM, septembre 2025), le projet couvre la modélisation de la base de données et le développement d'une application web en PHP natif avec base de données MySQL sous XAMPP.

Mission 1 — Base de données

  • MCD Looping — modélisation conceptuelle des données fournie, implantation de la base sous XAMPP
  • Implantation MySQL — création des tables et relations à partir du MCD
  • Jeux de données — alimentation de la base avec des données cohérentes pour les tests

Mission 2 — Application PHP

  • Page d'accueil (index.php) — navigation entre les pages (liens href) · Yasmin
  • Gestion des clients — CRUD complet : ajout, modification, suppression · Evan K.
  • Gestion des voitures — CRUD complet : ajout, modification, suppression · Samuel
  • Gestion des réservations — CRUD complet : ajout, modification, suppression · Mehmet
  • Authentification — partie connexion sécurisée · Axel
  • CSS & mise en page — design de l'application · Yasmin

Ma contribution — Gestion des voitures

J'ai développé en autonomie le module complet de gestion des voitures : la page d'affichage du catalogue et les quatre opérations CRUD en PHP natif avec requêtes MySQL via PDO.

R
Read — Affichage
Liste des voitures

Page principale affichant le catalogue de toutes les voitures disponibles en base : immatriculation, marque, modèle, kilométrage, statut. Requête SELECT avec affichage dynamique en tableau PHP.

C
Create — Ajout
Ajouter une voiture

Formulaire d'ajout d'une nouvelle voiture avec validation des champs côté PHP. Insertion en base via requête INSERT INTO préparée avec PDO.

U
Update — Modification
Modifier une voiture

Formulaire pré-rempli avec les données existantes de la voiture sélectionnée. Mise à jour en base via requête UPDATE après validation des nouvelles valeurs.

D
Delete — Suppression
Supprimer une voiture

Suppression d'une voiture via un bouton dédié avec confirmation, exécutant une requête DELETE ciblée par l'identifiant de la voiture.

Architecture technique

  • PHP natif — scripts PHP sans framework, un fichier par fonctionnalité
  • MySQL (XAMPP) — base de données locale, implantée à partir du MCD Looping
  • HTML / CSS — interface utilisateur
  • MCD — modélisation conceptuelle des données (Looping) : clients, voitures, réservations

Organisation du projet

  • Séances — 05/09 · 12/09 · 19/09 · Oral : 22/09 – 01/10/2025
  • Répartition — chaque membre responsable d'un module (clients, voitures, réservations, connexion, CSS)
